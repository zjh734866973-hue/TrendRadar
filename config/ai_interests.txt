# ===============================================================

# 总体目标

# ===============================================================

我希望 TrendRadar 帮我筛选和总结一份偏“情报简报”风格的信息流，而不是普通热搜合集。

请优先关注以下五类信息，并尽量保持大致比例：

1. AI 与科技趋势：约 30%
2. 中国时事、政策与产业变化：约 25%
3. 国际新闻与地缘政治：约 20%
4. 全球财经、金融市场与宏观经济：约 15%
5. 各平台高价值热搜与公共讨论：约 10%

如果某条新闻同时属于多个方向，请优先归入更靠前、更核心的类别。例如：一条“美国限制英伟达 AI 芯片出口中国”的新闻，应优先归入“AI 基础设施 / 芯片与地缘科技竞争”，而不是普通国际新闻或普通财经新闻。

# 5. 高价值热搜与公共讨论

# ===============================================================

热搜和热点不是主要信息来源，只作为舆情和社会关注度补充。

我关注的热搜类型包括：

1. 反映公共政策、社会治理、监管变化的热搜。
2. 反映消费者权益、平台治理、企业责任、产品安全的热搜。
3. 反映重大科技产品发布、AI 工具爆火、产业事件发酵的热搜。
4. 反映公众对经济、就业、教育、医疗、房价、收入等问题态度变化的热搜。

请降低以下内容优先级：

* 明星八卦、饭圈争议、恋情、综艺、娱乐营销。
* 纯情绪化社会新闻、猎奇新闻、低价值冲突。
* 促销、带货、抽奖、优惠券、游戏皮肤、影视宣发。
* 没有公共价值、产业价值或趋势判断价值的生活小热点。

# ===============================================================

# 6. 信息判断偏好

# ===============================================================

请优先保留具有以下特征的内容：

1. 能说明趋势变化
   例如技术路线变化、政策方向变化、市场预期变化、产业格局变化、组织战略变化。

2. 能影响决策
   例如 AI 工具选择、企业数字化转型、行业机会判断、技术学习方向、市场风险判断。

3. 能形成长期认知
   例如国家战略、产业竞争、技术范式变化、公司组织能力、商业模式变化。

4. 信息密度高
   优先选择事实明确、信源较强、数据较多、影响较大的新闻。
   低质量短标题、重复搬运、营销号改写、情绪化标题应降低优先级。

5. 适合摘要分析
   请优先选择能够被总结为“发生了什么、为什么重要、可能影响什么、我需要关注什么”的内容。

# ===============================================================

# 7. 分类优先级规则

# ===============================================================

如果一条新闻可以被分到多个类别，请按以下顺序分类：

1. AI、大模型、智能体、算力、芯片、机器人
2. 中国政策、产业、科技公司和社会治理
3. 国际关系、地缘政治和全球安全
4. 全球财经、金融市场和宏观经济
5. 高价值热搜和公共讨论
6. 文化、消费、娱乐、生活类内容

例如：

* “英伟达因美国出口限制调整中国 AI 芯片” → 优先归入 AI 基础设施 / 芯片与中美科技竞争。
* “DeepSeek 发布新模型并降低 API 价格” → 优先归入大模型与 AI 产品。
* “美联储释放降息信号推动科技股上涨” → 优先归入全球财经与宏观金融。
* “微博热议某公共政策调整” → 如果有政策意义，归入中国时事；如果只是情绪化讨论，降低优先级。

# ===============================================================

# 8. 标题质量要求

# ===============================================================

即使匹配了上面的兴趣方向，符合以下特征的标题也请跳过或降低优先级：

* 不要标题党、震惊体、夸张体，例如“震惊！”、“太可怕了！”、“彻底炸了！”、“刚刚！”、“万万没想到”。
* 不要营销软文、广告推广、课程引流、卖课、带货、优惠券内容。
* 不要明星八卦、饭圈争议、综艺炒作、影视宣发。
* 不要纯情绪宣泄、无明确事实、无可靠来源的内容。
* 不要重复度极高、只是不同平台搬运同一标题的内容。
* 不要缺少主体、缺少事件、缺少影响的信息。
* 不要只靠“网友热议”“全网沸腾”“评论区炸了”来制造热度的内容。

# ===============================================================

# 9. 摘要偏好

# ===============================================================

如果系统需要对新闻做 AI 总结，请尽量按以下方式理解和提炼：

1. 先说明事实：发生了什么。
2. 再说明重要性：为什么值得关注。
3. 再说明影响：可能影响哪些公司、行业、政策或市场。
4. 最后给出观察点：后续应该继续看什么信号。

我更需要“结构化判断”，不需要情绪化评论。
